# Backend Monorepo Template for Microservice Architecture

This repository provides a structured backend monorepo designed to support a microservice-based architecture.

---

## 🧩 Overview

- **backend/srv**  
  Contains individual microservices. Each service is self-contained and independently deployable.

- **backend/libs**  
  Shared libraries used across multiple services (utilities, common models, clients, etc.).

- **deployment**  
  Infrastructure and deployment-related configurations (Docker, Kubernetes, CI/CD, etc.).
